# Lonton

mi le lo wi no onin wija ani titan ihamo<br>
pi sekano je ta Tense ihamo le lo<br>
le sapi pi ani pa sila teka<br>
in sapi je uli, sapi je kujo.

pi ani sema tonko je ani nin,<br>
pi ani sema osole je menwo talun,<br>
pi ani mu kota, pi ani ihamo,<br>
men mi le sema in laki pan ihamo mijalo:

la, men sema tonko je pan naki je insu kali<br>
le lun kapan in ani insu hela je nelo pijen,<br>
un sema sipa je mu konkulen je nula tope<br>
le lo wen pani haja pi pelete je mu hukun.

lekin pi titan je nelo kin men mi le sema<br>
ta kota pan kapan je nin wonpa talun<br>
le lun lipan in pani kujo je menwo neso,<br>
le lun loso wija juka in lantan tiku je unitu.
